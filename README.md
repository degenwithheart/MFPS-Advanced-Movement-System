Create a new Unity project using URP (Universal Render Pipeline).
Make sure you are using a Unity version compatible with MFPS 2.0.

Import mfps-2.0-v1.10.0 (beta 3).unitypackage.
After importing, run the MFPS setup wizard and complete all required configuration steps (render pipeline, input system, etc.).

Import MFPS 1.10b3 – Advanced Movement.unitypackage.
This addon should be imported after the main MFPS package to ensure proper integration.
